# Clipboard Landing Page

## Overview

The Clipboard Landing Page is a responsive website designed to present the features of the Clipboard application. The project focuses on clean layout design, responsive structure, and modern front-end styling using HTML and CSS.

## Features

* Clean and modern landing page design
* Responsive layout for different screen sizes
* Organized sections highlighting product features
* Simple and user-friendly navigation

## Technologies Used

* HTML5
* CSS3

## Project Structure

* `index.html` – Main structure of the webpage
* `styles.css` – Styling and layout design
* `images/` – Project images and assets

## How to View the Project

1. Clone the repository
2. Open the project folder
3. Open `index.html` in your browser
